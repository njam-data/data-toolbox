New Jersey Advance Media Data Toolbox

A reporter repository for all things data, from basic instruction to complex visualization.

This page houses hundreds of databases, 64 analysis and visualization tools and 583 tutorials.






Welcome to the NJAM Data Toolbox. From this page, you can submit a request to the data team, find data and geography files for stories and find visualization tools and tutorials to help make your stories better. Scroll down and use the buttons below to find data, geographic mapping data, tutorials and tools available for use.

If you have questions about any of this, please e-mail Carla Astudillo or Stephen Stirling on the data desk.
